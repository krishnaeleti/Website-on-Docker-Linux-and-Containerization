Linux Docker Commands

In this section we will introduce some linux commands and how to log in the docker in linux.

Firstly, we will introduce some basic linux commands and how do they work.

  • cd /tmp/target

    Explanation: go to target dictionary path

  • cat ./project/Dockerfile

    Explanation: catch the content of the Dockerfile and print it in the console.

  • mv ./project/Dockerfile ./D1

    Explanation: move the Dockerfile in project folder to current folder and name it as D1

  • vim ./project/Dockerfile

    Explanation: use editor vim to edit the file

  • rm ./project/Dockerfile

    Explanation: remove the Dockerfile file

Figure 1: Docker based on the linux

Followings are the steps of setting up and starting a docker container in linux:

  • Step 1: Create Dockerfile and add content

    # vim ./Dockerfile

  • Step 2: Build dockerfile to an image

    # docker build --tag myDocker:1.0 .

  • Step 3: run a container from the image

    # docker run --name test1 myDocker:1.0
