Virtualization

What if you were able to run a computer within a computer? That is kind of the gist of what Virtualization is.

Virtualization is a good way to run multiple Operating Systems but using the same hardware resources. For example, a user owns a desktop computer running Windows Operating System, they have 8GB of RAM, 256GB of Hard drive Space and an 8 core CPU. With virtualization what they can do is create a virtual machine, and partition those resources into different Virtual machines and run other operating systems if they want to. It allows someone to have “2 computers” but using the same hardware resources.

In the image “Architecture of Virtualization’ you can see that there is a Host OS and then there is a hypervisor on top. Which is something that creates these VM’s and then on top of those there are Guest Operating Systems running their own program which the user dictates.
